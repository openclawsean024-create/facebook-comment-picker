# Facebook 留言抽獎器

一個可直接開啟 `index.html` 使用的單頁工具，專門用來做 **單一 Facebook 貼文留言抽獎**。

## 功能

- 支援單篇貼文留言名單抽獎
- 支援三種輸入格式：
  - `王小明`
  - `王小明 | 我要參加`
  - `王小明, 我要參加`
- 可設定中獎名額
- 可設定固定抽獎種子，讓結果可重現
- 可排除關鍵字（例如：測試、取消、機器人）
- 可設定是否「同名只保留一次」
- 一鍵複製中獎名單

## 使用方式

1. 直接用瀏覽器開啟 `index.html`
2. 輸入活動名稱（選填）
3. 貼上留言名單
4. 設定抽獎名額、去重規則、排除關鍵字
5. 按下「開始抽獎」

## 注意

由於 Facebook 留言資料抓取會受登入、隱私、CORS 與 Graph API 權限限制影響，這一版先專注在 **抽獎核心**：

- 你可以手動貼上留言
- 或把留言整理成 CSV / 純文字後匯入

如果要做成「輸入 Facebook 貼文網址 → 自動抓留言 → 抽獎」，下一步可再加：

- Facebook Graph API 串接
- 後端代理抓取
- 匯入/匯出 CSV
- 抽獎紀錄保存
- 黑名單與資格篩選
